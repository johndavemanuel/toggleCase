# Toggle Case
A collection of mixed case generation and text conversion tools

Inspired by http://www.togglecase.com/

### Made in
- Vue.js
- Tailwind CSS

### Includes
- Title Case
- Sentence Case
- Lower Case
- Remove Vowels
- Remove Consonants
- Camel Case
- Studly Caps
- Underscored Text
- Remove Duplicate Space
- Reverse Text
- Backward Text
- Scramble Text
- Encode URL
- Decode URL
- ROT13
